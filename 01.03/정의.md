Java -> class 단위

class
:  사용자가 정의하는 데이터 타입 (type). 여러 개의 데이터 종류를 하나의 타입으로 정의함.

여러 개의 데이터 -> 표로 구성되어 있음 => "클래스"

type
- int : 정수
- double : 소수
- char : 글자 하나
- boolean : 참/거짓
- string : 문자열

연산
: CPU를 움직인다.

new (연산자)
: 메모리 주소를 받아냄. 새로운 메모리 공간을 만듦.

변수 vs 배열
- 변수 : 하나의 값을 저장. 실행 중에 다른 작업 불가
- 배열 : 여러 개의 값을 저장. 실행 중에 다른 작업 가능

여러 개의 데이터를 처리할 때 => "배열"
